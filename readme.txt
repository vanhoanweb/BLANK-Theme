== Installation ==

1. Unzip, upload, activate, done (typical WP theme installation).
2. All done. Now go forth and use this starter theme to create something awesome.

For more infos and alternate install methods, check out the WP Codex:

https://codex.wordpress.org/Using_Themes#Using_Themes



== Upgrading ==

BLANK is a starter theme, which means it's designed for use as a starting point for building your own themes. Upgrading to the latest version is only recommended if you have not made any changes to the theme.
